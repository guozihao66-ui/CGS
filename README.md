# CGS Cobbler

CGS Cobbler is an internal consulting workflow prototype for CGS Advisors.

It helps consultants turn a new client brief into an engagement starter pack by activating proprietary frameworks, prior engagements, hypotheses, templates, and a first-draft recommendation workflow.
